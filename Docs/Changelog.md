## [1.0.0-beta] - 2025-11-23

### Добавлено

- ✅ Инициализация проекта SuspenseCore
- ✅ Модульная архитектура с 8 модулями:
  - SuspenseCore (ядро)
  - PlayerCore (системы игрока)
  - GAS (Gameplay Ability System)
  - EquipmentSystem (экипировка)
  - InventorySystem (инвентарь)
  - InteractionSystem (взаимодействия)
  - BridgeSystem (коммуникация между модулями)
  - UISystem (пользовательский интерфейс)
- ✅ Базовая структура каждого модуля (Public/Private)
- ✅ Build.cs конфигурация для всех модулей
- ✅ Plugin descriptor (SuspenseCore.uplugin)
- ✅ .gitignore для Unreal Engine проектов
- ✅ Comprehensive README.md с правилами разработки
- ✅ Структура документации (Docs/)
- ✅ Changelog для отслеживания изменений

### Техническое

- **Версия UE:** 5.7+
- **Язык:** C++
- **Стандарт кода:** UE Coding Standard
- **Архитектура:** Модульная, event-driven, replication-ready
