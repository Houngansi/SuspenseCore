# Legacy Modules Documentation

Documentation about legacy MedCom modules structure, metrics, and statistics.

## Contents

- **LegacyModulesDirectoryStructure.md** - Complete directory structure of all legacy modules
- **LegacyModulesMetricsAnalysis.md** - Code metrics and statistics analysis
- **LegacyModulesStatistics.md** - Aggregated statistics and summaries

## Purpose

This directory contains reference information about the original MedCom codebase:
- File organization and structure
- Lines of code, complexity metrics
- Module dependencies
- Code distribution analysis

## Usage

Use these documents to:
- Understand legacy code organization
- Reference original structure during migration
- Compare before/after metrics
- Identify migration scope and effort
